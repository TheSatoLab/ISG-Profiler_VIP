# ISG-Profiler_VIP

ISG Profiler & ISG-VIP: Identifying hidden viral infections across diverse animal species using innate immune signatures.

## Project structure

| Tool                            | description                                                                    |
| :------------------------------ | :----------------------------------------------------------------------------- |
| [ISG-Profiler](./isg-profiler/) | Cross-species ISG profiling for hidden virus detection in animal RNA-Seq data. |
| [ISG-VIP](./isg-vip/)           | Machine learning-based viral infection prediction using ISG profiles.          |
